## javafx-clojure-app
This program is for linux , Mac , And Windows
![](capture1.PNG)
## Installation

    $ git clone https://github.com/elect000/javafx-clojure-app/ jca
    $ cd jca
    
  1. from lein         
          
         $ lein run
  2. from jar
  
         $ lein uberjar
         $ cd target/uberjar
         $ java -jar *standalone.jar
